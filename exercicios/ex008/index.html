Principais formatações

Negrito ou Destaque

Nesta frase temos uma parte em negrito usando a tag B (Não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica).

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande usando a tag BIG, e um texto pequeno usando a tag SMALL nesse parágrafo.

Texto deletado

Podemos chamar um texto como excluído com a tag DEL, para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido com a tag INS para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Existe também o sublinhado como a tag U(não semântica).

Texto sobrescrito

Texto sobrescrito e usado para inserir coisas do tipo x20+3, usando a tag SUP

Texto subescrito

Texto subescrito e usado para inserir coisas do tipo CO2, usando a tag SUB
